年份 比赛项目 对阵双方 精彩描述
2008 羽毛球男单 1/16 决赛 林丹 VS 吴蔚 林丹展现出强大的实力，直落两局轻取对手，为后续的夺冠之路奠定基础
2008 羽毛球男单半决赛 林丹 VS 陈金 两位中国顶尖选手上演强强对话，林丹凭借出色的技术和稳定的发挥，成功战胜队友陈金，闯入决赛
2008 羽毛球女单 16 分之 1 决赛 张宁 VS 波萨那 老将张宁面对实力不俗的波萨那，凭借丰富的经验和顽强的斗志，成功拿下比赛，展现了其坚韧不拔的精神
2008 羽毛球男单决赛 林丹 VS 李宗伟 这是一场备受瞩目的巅峰对决，林丹在比赛中发挥出色，无论是进攻、防守还是控制节奏都占据上风，最终直落两局击败李宗伟，夺得冠军，展现了超强的统治力
2012 羽毛球男单半决赛 林丹 VS 李炫一 林丹在先失一局的不利局面下，连下两城逆转战胜李炫一，展现了其强大的心理素质和顽强的拼搏精神
2012 羽毛球男单决赛 林丹 VS 李宗伟 双方再次在决赛中相遇，比赛过程跌宕起伏，林丹在决胜局中顶住压力，最终以 2-1 的比分逆转获胜，成功卫冕奥运会男单冠军，这场比赛也成为了羽毛球历史上的经典之战136.
2012 羽毛球男双决赛 蔡赟 / 傅海峰 VS 鲍伊 / 摩根森 双方实力相当，比赛异常激烈，蔡赟 / 傅海峰配合默契，在关键时刻发挥出色，最终以 2-0 的比分击败对手，夺得男双冠军
2016 羽毛球男单决赛 谌龙 VS 李宗伟 谌龙在比赛中发挥出色，通过积极的进攻和多变的线路，有效地限制了李宗伟的发挥，最终直落两局战胜对手，夺得奥运会男单冠军，为中国羽毛球队再添一金
2016 羽毛球男双决赛 张楠 / 傅海峰 VS 吴蔚昇 / 陈蔚强 双方展开了一场激烈的强强对话，比赛过程精彩纷呈，张楠 / 傅海峰在首局失利的情况下，连下两局实现逆转，最终以 2-1 的比分夺得冠军，成功卫冕奥运会男双冠军，这场比赛也展现了他们顽强的斗志和过硬的技术.
